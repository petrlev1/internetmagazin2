Заголовки разделов и метатеги.

  • В поле "заголовок раздела" (title) введите заголовки соответствующего раздела, которыми он впоследствии и будет представлен.


  • В поле "мета теги (keywords)" введите ключевые слова (через запятую) по которым Вы хотели бы, чтобы данный ресурс находился поисковыми машинами.


  • В поле "мета теги (description)" введите (через запятую) словосочетания - также для поисковых машин.


* Примечание: Заголовки и метатеги важны для индексации в поисковых машинах, поэтому, чем подробнее они будут заполнены, тем больше вероятность нахождения данного ресурса в сети.

Редактирование текстовых разделов.

Текстовый раздел представляет собой информационное содержание с включением графических элементов. Редактирование данных разделов происходит в специальном редакторе, наподобие популярной программы MS-Word и подчиняется тем же правилам редактирования. Например, если Вы хотите включить рисунок в Ваш раздел, просто нажмите на кнопку "Image" в верней панели редактора и далее следуйте инструкциям, которые предложит система.

Для сохранения содержимого фрейма необходимо нажать на кнопку "save" верхней панели редактора, что подобно операции сохранения в программе MS-Word.

* Важно! При копировании информации из других источников (MS Word, другие сайты) не забывайте использовать опцию "Чистая вставка из Word" интерфейса веб редактора, т.к. при этом могут вставится и специфические правила разметки другого источника (например - лишний html код, который может быть не виден, но иметь свои правила верстки).

* Примечание: не рекомендуется изменять существующую верстку разделов (менять, либо удалять рисунки, увеличивать размер шрифта), т.к. это может привести к нарушению целостности дизайна раздела.

Все разделы сайта, доступные для редактирования, располагаются в левой части программы администрирования. Щелкните на нужный пункт, чтобы перейти к редактированию страницы.

Управлением каталогом

  • Создание разделов каталога. Для добавления нового раздела каталога зайдите по ссылке "Каталог" из левого меню программы администрирования. В открывшемся интерфейсе, в поле "Название раздела/подраздела/объекта" введите название Вашего раздела и нажмите кнопку "Сохранить", после этого Ваш раздел появится в списке объектов.

  • Создание подразделов. В поле "Название раздела/подраздела/объекта" введите название Вашего подраздела, а из выпадающего списка поля "Создать подраздел" выберите раздел, для которого требуется создать подраздел. После этого нажмите кнопку "Сохранить" и Ваш подраздел появиться в списке объектов ниже названия раздела.

  • Добавление объектов каталога. Для добавления объекта в каталог, в поле "Название раздела/подраздела/объекта" введите наименование объекта, а из выпадающего списка поля "Создать подраздел/объект" выберите подраздел, в который требуется добавить объект и нажмите кнопку "Сохранить". После этого объект появится в списке объектов представленный ссылкой, кликните мышкой на объект, после чего Вам откроется интерфейс редактирования объекта.

Пароли

Для того чтобы сменить пароли к системе администрирования сайта, необходимо выбрать опцию "Сменить пароли" раздела "Пароли". В появившихся полях "Логин" и "Новый пароль" введите соответственно новые логин и пароль, после чего нажмите кнопку "Создать". После установки новых значений, программа администрирования будет автоматически перезагружена, для входа в программу введите новые значения логина и пароля.

Для выхода из системы администрирования нажмите на ссылку "Выход" расположенную в верхнем правом углу программы.

* Важно! Длина создаваемых логина и пароля не должна превышать 25 символов. Не используйте символы пробелов, кавычек и русских букв в названиях создаваемых паролях, все остальные символы допускаются.
